# pdf-extracter
Python提取pdf中注释和高亮内容的脚本，方便阅读论文以后对注释进行汇总。以前为了图方便就在pdf标了注释，写论文的时候简直折磨，干脆写了个脚本提高效率。代码参考来自https://blog.csdn.net/jaytalent/article/details/112471396|
运行前配置：
pip install PyPDF2 |
pip install fitz |
pip install PyMuPDF==1.16.14
